# Fonts

Drop these `.ttf` files here before running the app. All are free on Google Fonts.

- `BebasNeue-Regular.ttf` — https://fonts.google.com/specimen/Bebas+Neue
- `PlayfairDisplay-Regular.ttf`, `PlayfairDisplay-Bold.ttf` — https://fonts.google.com/specimen/Playfair+Display
- `Inter-Regular.ttf`, `Inter-Bold.ttf` — https://fonts.google.com/specimen/Inter
